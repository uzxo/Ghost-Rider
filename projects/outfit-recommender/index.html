Outfit Recommendation System
Weather × Fashion rules × ML ranking • R + Bash + ETL
← Back to Portfolio

A production-style pipeline that suggests daily outfits by combining live/queried weather, a rules engine (e.g., temperature-aware layers, fabric constraints), and an ML ranker. Includes ETL for catalog data, an R API for inference, and a bash runner to orchestrate the workflow.

Read Executive Summary (PDF) View Code on GitHub Preview Plot

How it works

  • ETL: etl.R ingests and cleans catalog + metadata.
  • Weather: weatherstack_api.R pulls conditions used as features.
  • Recommender: ootd_api.R applies fashion rules & ML ranking.
  • Automation: run_pipeline.sh orchestrates the end-to-end job.
  • Scraping: product_scraping.R builds/refreshes the item catalog.

Quick start (local)

# Install R deps in your environment, then: # 1) ETL Rscript etl.R # 2) (Optional) scrape/refresh catalog Rscript product_scraping.R # 3) Run API/pipeline locally bash run_pipeline.sh # or Rscript run_ootd_api.R

Note: GitHub Pages cannot execute R/bash. Run locally or in a server environment.
